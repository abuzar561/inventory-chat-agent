# inventory-chat-agent
An AI-powered inventory chat agent built with n8n that updates and manages stock in Google Sheets automatically through natural language commands.


 AI Inventory Chat Agent (Built with n8n)
This project is part of my AI Automation series. It’s an AI-driven workflow built entirely in n8n, designed to manage inventory through chat commands.

You can talk to the bot naturally, and it will automatically update or fetch inventory data from Google Sheets — no manual entry needed!

... Example Commands: ...

“Add 10 phones to stock”

“Update T-shirts to 25 pieces”

“Check how many laptops are left”

... How It Works: ...

Uses OpenAI API (or any LLM) to understand chat messages

Uses n8n workflows to connect logic and APIs

Integrates with Google Sheets API for live inventory updates
